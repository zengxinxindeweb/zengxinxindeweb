MAC hexo 搭建笔记

Veröffentlicht am 2016-08-29   |  

安装hexo

sudo npm install -g hexo

初始化

终端cd到一个选定的目录

cd /**/**

然后

hexo init blog

blog是建立的文件夹名称。cd到blog文件夹下，执行如下命令，安装npm：

npm install

执行命令，开启hexo服务器：

hexo s  或者  hexo server

浏览器中打开网址http://localhost:4000 进行测试！

_config.yml
打开命令： vim _config.yml
修改：

deploy:
  type: git
  repository:https://zengxinxinde/zengxinxinde/gonghonglou.github.io.git
  branch: master

修改好后esc然后切换小写输入命令：:wq!

清除缓存文件：hexo clean

在blog文件夹目录下执行生成静态页面命令：

hexo g  或者  generate

在blog文件夹目录下执行配置命令：

hexo d  或者  hexo deploy

同步生成静态页面与执行配置：hexo d -g
